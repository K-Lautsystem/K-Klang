K‑KLANGALPHABET – MINIMALER SYSTEMCODE

1. STIMMEN (7 Energieformen)

A = Ursprung / Atem
S = Söl‑Stimme (erste Spannung)
U = Sun‑Stimme (Resonanz)
C = Chi‑Stimme (Impuls, Druck)
BS = Ban‑Söl (halb Spannung, halb Resonanz)
BC = Ban‑Chi (halb Impuls, halb Spannung)
H = Hu‑Stimme (tiefe Resonanz)

2. ACHSEN (3 Formdimensionen)

E0–E4 = Energie‑Achse
V0–V4 = Vertikal‑Achse
H0–H4 = Horizontal‑Achse

3. BANJÖL‑PHASEN (3 Klangphasen)

B0 = Startimpuls
B1 = Resonanzkern
B2 = Abschlussimpuls

4. K‑LAUT‑CODE (offizielle Struktur)

[STIMME]-E<V>-V<V>-H<V>-B<V>

5. BEISPIELE

U‑E1V4H2‑B1 = weicher Vokal
C‑E3V1H4‑B0 = harter Anfang
BS‑E1V2H1‑B2 = nasaler Endlaut
